## Executive Objective

ER-WEB-005 provides the canonical public engineering handbook for the Enterprise Reality Website. Part I is published as the approved foundation for the remaining handbook Parts.

## Scope

This foundation owns publication structure, governance, master metadata, Part catalogues, the chapter catalogue, diagram governance, asset governance, workflow, versioning and review rules.

It does not author technical chapters, create architecture diagrams, add screenshots, provide code examples, change website routes, change site implementation, alter deployment or change DNS.

## Intended Audience

The intended audience is website engineers, publication authors, technical reviewers, architecture reviewers, accessibility reviewers, release reviewers and programme approvers.

## Reading Guide

Read this file first for the publication objective and table of contents. Read `CONSTITUTION.md` for governing authority, `PUBLICATION-INDEX.md` for chapter management, `CHAPTER-METADATA-TEMPLATE.md` before creating chapters, and the relevant Part README before authoring any chapter file.
