# Chapter 3 - Constitutional Thinking

## 1. Purpose

### Chapter Purpose

Establish the constitutional principles that govern the website.

### Intended Learning Outcomes

- Recognise Truth Before Marketing as a governing principle.
- Recognise Static Before Dynamic as a governing principle.
- Recognise Architecture Before Technology as a governing principle.
- Recognise Documentation as Code as a governing principle.
- Recognise Progressive Enhancement as a governing principle.
- Recognise Accessibility by Design as a governing principle.
- Recognise Explainability as a governing principle.
- Recognise Governed Publication as a governing principle.
- Recognise the Public/Private Boundary as a governing principle.

### Chapter Boundary

This chapter identifies approved governing principles for later authoring. It does not interpret, expand or amend those principles beyond the approved publication set.
