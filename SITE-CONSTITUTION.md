# Site Constitution

This repository is governed by
[ER-CON-002](publications/ER-CON-002-Enterprise-Reality-Website-Constitution.md),
the canonical Enterprise Reality Website Constitution.

This file is a repository-facing working summary. It does not replace,
amend or reinterpret ER-CON-002.

## Working Principles

- Tell the truth before marketing the programme.
- Publish only approved public material.
- Distinguish released, in-development, planned, research and concept
  capabilities.
- Preserve the separation between the public website repository and the
  private Enterprise Reality platform repository.
- Prefer static, portable, accessible implementation.
- Avoid unnecessary data collection, third-party scripts and speculative
  product claims.
- Treat accessibility, privacy, security and publication governance as
  implementation requirements.

## Repository Boundary

This repository may contain the public website implementation, approved public
publications, public documentation and supporting quality controls.

It must not contain private platform source, internal prompts, unpublished
reviews, confidential correspondence, employer or client material, personal data,
credentials, secrets or unapproved drafts intended for publication.

## Authority

When this summary conflicts with ER-CON-002, ER-CON-002 prevails. When
implementation convenience conflicts with the approved website publications, the
approved publications prevail.
